# 🏦 Bank Management System (Java - Console Based)

A simple console-based banking application built using **Java**, designed to simulate basic banking operations through a command-line interface.

---

## 🚀 Features

* 📊 View account balance
* 💰 Deposit money
* 💸 Withdraw funds
* ⚠️ Input validation for secure transactions
* 🔄 Continuous user interaction via menu system

---

## 🛠️ Technologies Used

* Java (Core Java)
* Object-Oriented Programming Concepts

---

## 📚 Concepts Demonstrated

This project showcases key programming fundamentals:

* ✔️ Methods and modular code structure
* ✔️ Loops (for continuous operation)
* ✔️ Conditional statements (decision making)
* ✔️ User input handling (Scanner class)
* ✔️ Basic error handling and validation

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/FelixAluchio/bank-management-system.git
   ```

2. Navigate to the project folder:

   ```bash
   cd bank-management-system
   ```

3. Compile the program:

   ```bash
   javac Main.java
   ```

4. Run the application:

   ```bash
   java Main
   ```
